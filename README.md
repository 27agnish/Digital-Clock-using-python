# Digital-Clock-using-python
A simple and visually appealing digital clock desktop application built using Python and PyQt5. This project displays the current system time in real-time with a clean UI and a classic digital clock aesthetic.

# 🚀 Features
* 🕒 Real-time clock display (hours, minutes, seconds)
* 🔁 Automatic update every second using QTimer
* 🎨 Custom UI styling (black background + neon green text)
* 🔤 Custom digital font (DS-DIGIT) for authentic clock look
* 📐 Centered and responsive layout using QVBoxLayout
* 🖥️ Lightweight and beginner-friendly PyQt5 project
## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** PyQt5
- **Concepts Used:**
  - GUI development
  - Event-driven programming
  - Timer-based updates
  - Custom font integration
# ⚙️ Installation & Run
* Clone the repository
-> git clone https://github.com/27agnish/Digital-Clock-using-python.git
*  Navigate to project folder
-> cd Digital-Clock-using-python
*  Install dependencies
-> pip install PyQt5
*  Run the application
-> python main.py
# 🧠 How It Works
* The application uses a QLabel widget to display the time
* A QTimer updates the time every second
* The system time is fetched and formatted into a readable string
* Styling is applied using Qt Style Sheets for a modern digital look
